## ROS2

Profile: **ros2**

The `ros2` profile describes how to create mcap files for v2 of the [ROS](https://ros.org/) framework.

### Channel Info

| field       | value            |
| ----------- | ---------------- |
| encoding    | cdr              |
| schema      | .MSG             |
| schema_name | package/datatype |
| user_data   | See below        |

#### user_data

- offered_qos_profiles (required, string)
