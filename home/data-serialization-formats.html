Data Serialization Formats

MCAP files can store multiple streams of heterogeneous data, serialized in any of the following data formats:

  • ROS 1
  • ROS 2
  • JSON Schema
  • Protobuf
  • FlatBuffers

To start reading and writing MCAP messages in these formats, check out the Python and C++ tutorials in our Guides section.
